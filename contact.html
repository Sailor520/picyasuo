Technical Support

If you're experiencing technical issues, please include the following information in your email to help us assist you better:

Browser Information:

  • Browser name and version (Chrome, Firefox, Safari, etc.)
  • Operating system (Windows, macOS, iOS, Android)
  • Device type (desktop, mobile, tablet)

Issue Details:

  • Description of the problem you're experiencing
  • Steps you took before the issue occurred
  • Error messages (if any)
  • Image format and approximate file size you were trying to compress
