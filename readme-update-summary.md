### 4. 使用指南和支持格式

#### 输入格式支持
- **JPEG/JPG** ✅ 完全支持
- **PNG** ✅ 完全支持（含透明度）
- **WebP** ✅ 完全支持
- **GIF** ⚠️ 转换为JPEG（动画丢失）
- **SVG** ⚠️ 转换为JPEG

#### 输出格式
- **JPEG** - 适合照片，文件更小
- **PNG** - 适合透明图形
- **WebP** - 现代格式，压缩效果最佳
