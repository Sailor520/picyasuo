# PicYaSuo - 在线图像压缩工具

一个现代化的在线图像压缩工具，支持单图和批量处理，采用苹果风格设计，提供优质的用户体验。

## ✨ 主要功能

### 🖼️ 图像压缩
- **单图模式**: 上传单张图片，实时预览压缩效果
- **批量模式**: 支持同时处理最多30张图片
- **智能模式切换**: 根据上传文件数量自动切换处理模式
- **质量控制**: 可调节压缩质量（1%-100%）
- **多格式支持**: JPG、PNG、WebP等主流格式

### 📦 批量处理优化
- **紧凑布局**: 优化图片卡片尺寸，减少视觉占用
- **固定滚动区域**: 600px高度的固定区域，支持流畅滚动
- **ZIP打包下载**: 真正的ZIP压缩包下载，而非逐个文件
- **智能命名**: 压缩文件按序号命名，便于管理
- **进度跟踪**: 实时显示压缩进度和状态

### 🎨 用户界面
- **Apple风格设计**: 现代化的UI设计
- **响应式布局**: 完美适配桌面和移动设备
- **流畅动画**: 平滑的交互动画效果
- **直观操作**: 拖拽上传、一键压缩、批量操作

### 🔒 隐私安全
- **本地处理**: 所有图片处理都在浏览器中完成
- **无服务器上传**: 图片不会上传到任何服务器
- **内存管理**: 自动清理临时文件和URL对象

## 🚀 快速开始

### 环境要求
- Node.js 16.0.0 或更高版本
- 现代浏览器（支持ES6+）

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```
访问: http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 📱 使用方法

### 单图处理
1. 点击上传区域或拖拽单张图片
2. 调节压缩质量滑块
3. 预览压缩效果
4. 下载压缩图片

### 批量处理
1. 选择多张图片（2-30张）
2. 自动进入批量模式
3. 设置统一的压缩质量
4. 点击"压缩全部"进行批量压缩
5. 使用"打包下载"获取ZIP压缩包
